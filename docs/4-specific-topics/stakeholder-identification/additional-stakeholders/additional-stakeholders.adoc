== Additional Stakeholders

[abstract]
Stakeholders discovered later as the team learns more about the domain.

[cols="2,3,2,3",options="header"]
|===
|Stakeholder |Goals / needs |Influence |Concerns / notes

|Landlords / Property Managers
|Ensure that tenant households are kept in good order and that shared spaces are not overloaded or misused.
|Low
|Unclear lines between personal and shared household property; no formal mechanism for landlord oversight.

|Grocery / Retail Stores
|Attract repeat purchases by students and household groups; provide accurate product data (barcodes, pricing) that inventory systems may rely on.
|Low
|Incorrect or inconsistent product metadata in retail databases causing failed barcode lookups.

|Mobile Platform Reviewers (App Store / Google Play)
|Ensure the mobile application complies with platform guidelines regarding user privacy, data handling, and interface standards before public distribution.
|High
|App rejection due to improper handling of user data or failure to meet minimum functionality requirements for listing.
|===

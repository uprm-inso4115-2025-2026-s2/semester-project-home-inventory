== Overview

Users can view inventories shared with them by other members.
Each shared inventory shows the user's role and limits actions accordingly.

== User Flow

....
[My Inventories] --> [Shared Inventory Detail] --> [Item Detail / Edit]
....

== Role Permissions

....
Action              | Owner | Editor | Viewer
--------------------|-------|--------|-------
View items          |  yes  |  yes   |  yes
Add / edit items    |  yes  |  yes   |  no
Delete items        |  yes  |  yes   |  no
Manage members      |  yes  |  no    |  no
....

== Flow Annotations

=== Navigating to a Shared Inventory

. User lands on My Inventories.
. Shared inventories appear under *Shared With Me* with a role badge.
. User taps a card to open the detail view.
. Role is shown in the nav bar throughout.

=== How Role Affects Actions

. *Viewer* — browse only. Add, Edit, and Delete are hidden.
. *Editor* — can add, edit, and delete items. Cannot manage members.
. *Owner* — full access including managing members.
